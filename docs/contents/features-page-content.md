## 1. 📅 예약 관리

### 섹션 타이틀
```
예약 관리
모든 예약을 한곳에서
```

### 기능 상세

**1-1. 예약 캘린더**
```
[대형 스크린샷]

📅 일/주/월 뷰로 자유롭게 전환
- 오늘 예약만 보기
- 이번 주 전체 보기
- 한 달 예약 현황 파악

🎨 디자이너별 색상 구분
- 김디자이너: 파란색
- 이디자이너: 초록색
- 한눈에 누가 바쁜지 확인

🖱️ 드래그로 예약 시간 조정
- 시간 변경도 드래그 한 번
- 클릭으로 상세 정보
- 더블클릭으로 빠른 수정

실제 사용 후기:
"달력만 봐도 오늘 하루가 한눈에!
더 이상 수첩 안 봐요."
```

**1-2. 24시간 자동 예약 접수**
```
[고객 예약 화면 스크린샷]

🌙 밤에도 새벽에도 예약 접수
- 고객이 언제든 직접 예약
- 관리자는 아침에 확인만
- 전화 받으러 뛰어갈 필요 없음

📱 모바일 최적화
- 스마트폰에서 편리하게
- 카카오톡으로 링크 공유
- yourshop.moer.io 전용 주소

⚡ 즉시 확인 가능
- 예약 즉시 대시보드 반영
- 푸시 알림 (선택)
- 이메일 알림 (선택)

실제 사용 후기:
"자면서도 예약이 들어와요.
아침에 일어나면 이미 5건!"
```

**1-3. 시간 충돌 자동 방지**
```
[충돌 방지 화면 스크린샷]

⚠️ 예약 겹침 걱정 Zero
- 이미 예약된 시간은 선택 불가
- 시스템이 자동으로 체크
- 사람 실수 완벽 차단

⏱️ 시술 시간 자동 계산
- "컷" 선택 → 60분
- "컷+염색" 선택 → 120분 자동
- 다음 예약 가능 시간 자동 표시

🚫 휴무 시간 자동 제외
- 점심시간 (12-1시) 예약 불가
- 저녁 휴식 (6-7시) 예약 불가
- 공휴일/특별 휴무 자동 반영

실제 사용 후기:
"예약이 겹쳐서 난감했던 적
이제 한 번도 없어요!"
```

**1-4. 예약 상태 관리**
```
[상태 관리 화면 스크린샷]

📊 예약 상태별 분류
- 대기중 (확인 필요)
- 확정 (승인 완료)
- 완료 (시술 끝)
- 취소 (노쇼/고객 취소)

✅ 원클릭 승인/거절
- 예약 클릭 → 승인 버튼
- 카카오톡 자동 발송
- 3초면 처리 완료

📝 예약별 메모
- "VIP 고객"
- "염색약 알러지 주의"
- "다음에 펌 추천"

실제 사용 후기:
"메모 기능 덕분에
특별 요청 절대 안 까먹어요."
```

## 3. 👥 고객 관리

### 섹션 타이틀
```
고객 관리
단골 고객, 이제 한눈에
```

### 기능 상세

**3-1. 고객 자동 등록**
```
[고객 등록 화면 스크린샷]

📝 예약 시 자동 생성
- 전화번호만 입력
- 고객 정보 자동 저장
- 별도 등록 작업 불필요

🔍 중복 체크
- 같은 전화번호 자동 인식
- "기존 고객입니다" 표시
- 이력 바로 확인 가능

⚡ 빠른 검색
- 이름 일부만 입력
- 전화번호 뒷자리로 검색
- 1초 안에 찾기

실제 사용 후기:
"고객 찾는 게 너무 빨라요.
수첩 넘기던 시간 절약!"
```

**3-2. 방문 이력 자동 기록**
```
[이력 화면 스크린샷]

📊 모든 방문 자동 저장
- 방문 날짜
- 시술 내역
- 담당 디자이너
- 결제 금액

📈 자동 집계
- 총 방문 횟수
- 총 결제 금액
- 평균 시술 간격
- 최근 방문일

🎯 맞춤 서비스 가능
- "지난번에 OO 하셨죠?"
- 이력 보고 자연스럽게 대화
- 고객 만족도 up

실제 사용 후기:
"손님이 '기억해주시네요'
라며 좋아해요!"
```

**3-3. 고객 태그 관리**
```
[태그 화면 스크린샷]

🏷️ 자유롭게 태그 추가
- VIP
- 단골
- 민감성 두피
- 염색 알러지
- 재방문 확률 높음

🎨 색상으로 구분
- VIP: 골드
- 단골: 블루
- 주의: 레드

🔍 태그로 필터링
- "VIP 고객만 보기"
- "알러지 고객 확인"
- 이벤트 문자 발송 시 활용

실제 사용 후기:
"VIP 고객 따로 관리하니까
특별 서비스 제공 편해요!"
```

**3-4. 고객 메모**
```
[메모 화면 스크린샷]

📝 중요한 정보 기록
- "다음에 펌 추천"
- "어머니 소개로 방문"
- "염색약 No.5 사용"
- "대화 많이 원치 않음"

🔒 비공개 메모
- 관리자만 볼 수 있음
- 고객은 모름
- 내부 인수인계용

⏰ 시간별 메모
- 방문할 때마다 추가
- 히스토리 확인 가능

실제 사용 후기:
"메모 덕분에 특별 요청
절대 안 까먹어요!"
```
