### FAQ 아코디언 (10개)

**Q1. 설치가 어렵나요?**
```
A. 전혀 어렵지 않아요!

1. 회원가입 (1분)
2. 매장 정보 입력 (2분)
3. 디자이너 추가 (1분)
4. 시술 메뉴 추가 (1분)

5분이면 바로 사용 가능합니다.
복잡한 설정은 없어요!
```

**Q2. 카카오톡 알림은 어떻게 설정하나요?**
```
A. 매우 간단해요!

카카오 비즈니스 계정만 있으면 됩니다.
(무료로 만들 수 있어요)

moer 대시보드에서
카카오 채널 연결 → 완료!

설정 가이드 영상도 제공해드려요.
```

**Q3. 고객이 예약하는 페이지 주소는?**
```
A. yourshop.moer.io 형식이에요.

예: gangnam-hair.moer.io

매장명으로 자동 생성되며,
원하시면 변경도 가능해요.

카카오톡, 인스타에 링크만 공유하면
고객이 바로 예약할 수 있어요!
```

**Q4. 무료 체험은 정말 무료인가요?**
```
A. 네, 100% 무료입니다!

- 신용카드 등록 불필요
- 7일 동안 모든 기능 사용 가능
- 체험 종료 후 자동 과금 없음
- 원하시면 유료 플랜으로 전환

안심하고 체험해보세요!
```

**Q5. 계약 기간이 있나요?**
```
A. 없습니다!

- 월간 결제: 언제든 해지 가능
- 연간 결제: 남은 기간 환불

위약금도 없고, 해지도 클릭 한 번이에요.
마음 편하게 시작하세요!
```

**Q6. 모바일에서도 되나요?**
```
A. 물론이죠!

관리자 페이지도 모바일 최적화
고객 예약 페이지도 모바일 완벽 지원

스마트폰만 있으면 어디서든
예약 확인, 승인, 통계 확인 가능해요!
```

**Q7. 데이터는 안전한가요?**
```
A. 100% 안전합니다!

- 네이버 클라우드 인프라 사용
- SSL 암호화 적용
- 정기 백업 (자동)
- 개인정보보호법 준수

20년 경력 개발자가 직접 관리해요.
```

**Q8. 여러 매장을 운영하는데 가능한가요?**
```
A. 가능해요!

각 매장마다 별도 계정 생성
각각 독립적으로 운영 가능

다수 매장 할인은 별도 문의해주세요.
```
