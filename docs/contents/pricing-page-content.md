```
📉 다운그레이드도 가능

├─ 다음 결제일부터 적용
├─ 현재 결제 기간 끝까지는 프로 기능 사용

베이직 → 무료
├─ 다음 결제일부터 적용
├─ 현재 결제 기간 끝까지는 베이직 기능 사용

주의사항
├─ 다운그레이드 시 데이터는 유지
├─ 단, 제한 사항은 즉시 적용
└─ 예: 프로→베이직 시 재방문 알림 중단
```

## ❓ 요금제 FAQ

### 자주 묻는 질문

**Q1. VAT는 별도인가요?**
```
A. 네, VAT 10%가 별도 부과됩니다.

베이직 29,000원 + VAT 2,900원 = 31,900원
프로 59,000원 + VAT 5,900원 = 64,900원

사업자 고객은 세금계산서 발행 가능합니다.
```

**Q2. 7일 무료 체험은 어떤 플랜인가요?**
```
A. 베이직 또는 프로 플랜을 선택하여 체험 가능합니다.

- 7일 동안 선택한 플랜의 모든 기능 사용
- 신용카드 등록 불필요
- 체험 종료 후 자동 과금 없음
- 원하시면 유료 플랜으로 전환

무료 플랜은 체험 없이 바로 사용 가능합니다.
```

**Q3. 예약 건수를 초과하면 어떻게 되나요?**
```
A. 자동으로 차단되지 않습니다.

무료 플랜 (월 50건)
├─ 50건 초과 시 알림
├─ 추가 예약 접수 가능 (단, 경고 표시)
└─ 다음 달부터는 베이직 업그레이드 권장

베이직 플랜 (월 200건)
├─ 200건 초과 시 알림
├─ 추가 예약 접수 가능
└─ 다음 달부터는 프로 플랜 권장

프로 플랜
└─ 무제한 (제한 없음)
```

**Q4. 디자이너 수를 초과하면?**
```
A. 업그레이드가 필요합니다.

무료: 1명 → 베이직: 3명 → 프로: 무제한

베이직 플랜에서 4명째 디자이너 추가 시도 시
"프로 플랜으로 업그레이드하세요" 안내
```

**Q5. 환불 정책은 어떻게 되나요?**
```
A. 공정하게 처리합니다.

7일 무료 체험 기간
└─ 100% 환불 (이용 기간 상관없이)

유료 전환 후
├─ 사용 일수 차감 후 일할 계산 환불
├─ 예: 월 29,000원, 10일 사용 → 19,000원 환불
└─ 환불 신청 후 5영업일 이내 처리

연간 결제
├─ 남은 개월 수만큼 환불
└─ 예: 12개월 중 3개월 사용 → 9개월치 환불
```

**Q6. 플랜을 변경하면 데이터는 어떻게 되나요?**
```
A. 모든 데이터는 그대로 유지됩니다.

업그레이드 (무료 → 베이직 → 프로)
└─ 모든 데이터 유지 + 추가 기능 사용 가능

다운그레이드 (프로 → 베이직 → 무료)
├─ 모든 데이터 유지
├─ 단, 제한 사항 적용
└─ 예: 베이직→무료 시 고객 이력은 유지되지만 최근 10건만 표시
```

**Q7. 여러 매장을 운영하는데 할인이 있나요?**
```
A. 엔터프라이즈 플랜을 이용하세요.

10개 매장 이상
├─ 별도 견적
├─ 매장당 할인 적용
├─ 통합 관리 대시보드
└─ 전담 계정 매니저

문의: enterprise@moer.io
```

**Q8. 계약 기간이 있나요?**
```
A. 없습니다!

월간 결제
└─ 언제든 해지 가능, 위약금 없음

연간 결제
├─ 중도 해지 가능
└─ 남은 기간 환불

마음 편하게 시작하세요!
```

**Q9. 카카오톡 알림 비용은 별도인가요?**
```
A. 플랜에 포함되어 있습니다.

베이직 플랜
└─ 월 200건 포함 (초과 시 건당 50원)

프로 플랜
└─ 무제한 무료

대부분의 경우 추가 비용 없이 사용 가능합니다.
```

**Q10. 이미지 저장 용량은?**
```
A. 플랜별로 다릅니다.

무료 플랜
└─ 100MB (포트폴리오 5장 정도)

베이직 플랜
└─ 월 5GB (포트폴리오 약 1,000장)

프로 플랜
└─ 무제한

초과 시 추가 용량 구매 가능 (1GB당 월 1,000원)
```
