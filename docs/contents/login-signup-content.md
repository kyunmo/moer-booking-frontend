## 🔐 로그인 페이지 (`/login`)

### 페이지 제목
```
moer 로그인
```

### 로고 & 환영 메시지
```
[moer 로고]

환영합니다!
```

### 로그인 폼

```
─────────────────────────────
이메일
[                           ]

비밀번호
[                           ]

[ ] 로그인 상태 유지
─────────────────────────────

[로그인]

비밀번호를 잊으셨나요?
```

### 소셜 로그인

```
─────────────────────────────
또는 간편 로그인
─────────────────────────────

[Google로 로그인]
[Naver로 로그인]
[Kakao로 로그인]
```

### 하단 링크

```
아직 계정이 없으신가요?
[회원가입 하기 →]
```

### 폼 검증 메시지

**이메일 검증**
```
❌ 이메일 형식이 올바르지 않습니다
❌ 이메일을 입력해주세요
```

**비밀번호 검증**
```
❌ 비밀번호를 입력해주세요
❌ 비밀번호가 일치하지 않습니다
```

**로그인 실패**
```
❌ 이메일 또는 비밀번호가 올바르지 않습니다
❌ 계정이 존재하지 않습니다
❌ 계정이 정지되었습니다. 고객센터로 문의해주세요
```

**로그인 성공**
```
✅ 로그인되었습니다
→ /dashboard로 리다이렉트
```

## ✍️ 회원가입 페이지 (`/signup`)

### 페이지 구조 (4단계)

---

### Step 1: 플랜 선택

```
[moer 로고]

moer 시작하기

어떤 플랜으로 시작하시겠어요?
```

### 플랜 카드 (3개)

```
┌──────────┐  ┌──────────┐  ┌──────────┐
│   무료    │  │ 베이직 ⭐ │  │   프로    │
│           │  │           │  │           │
│  0원/월   │  │ 29,000원 │  │ 59,000원 │
│           │  │   /월     │  │   /월     │
│           │  │           │  │           │
│ • 50건/월 │  │ • 200건  │  │ • 무제한  │
│ • 1명     │  │ • 3명     │  │ • 무제한  │
│ • 광고 O  │  │ • 카톡 O  │  │ • 고급    │
│           │  │ • 광고 X  │  │   기능    │
│           │  │           │  │           │
│ [ 선택 ]  │  │ [ 선택 ]  │  │ [ 선택 ]  │
└──────────┘  └──────────┘  └──────────┘

💡 플랜은 나중에 언제든 변경 가능합니다
```

---

### Step 2: 계정 정보

```
계정 정보 입력
```

**이메일 가입 폼**

```
─────────────────────────────
이메일 *
[                           ]
💡 로그인 시 사용됩니다

비밀번호 * (8자 이상)
[                           ]

비밀번호 확인 *
[                           ]
─────────────────────────────

[ ] 이용약관에 동의합니다 (필수)
[ ] 개인정보처리방침에 동의합니다 (필수)
[ ] 마케팅 정보 수신에 동의합니다 (선택)

[다음 단계 →]
```

**또는 간편 가입**

```
─────────────────────────────
또는 간편 가입
─────────────────────────────

[Google로 계속하기]
[Naver로 계속하기]
[Kakao로 계속하기]

💡 간편 가입 시 매장 정보만 입력하면 끝!
```

### 폼 검증 메시지

**이메일**
```
❌ 이메일 형식이 올바르지 않습니다
❌ 이메일을 입력해주세요
❌ 이미 사용 중인 이메일입니다
✅ 사용 가능한 이메일입니다
```

**비밀번호**
```
❌ 비밀번호는 8자 이상이어야 합니다
❌ 비밀번호가 일치하지 않습니다
✅ 비밀번호가 일치합니다
```

**동의**
```
❌ 필수 약관에 동의해주세요
```

---

### Step 3: 매장 정보

```
매장 정보 입력
```

**기본 정보**

```
─────────────────────────────
매장명 *
[                           ]
💡 고객에게 노출되는 이름입니다

업종 *
[ 미용실 ▼ ]
  • 미용실/헤어샵
  • 네일샵
  • 필라테스/요가
  • 스터디카페
  • 공방
  • 기타

대표자명 *
[                           ]

전화번호 *
[                           ]
💡 010-1234-5678
─────────────────────────────

[이전]  [다음 단계 →]
```

### 폼 검증 메시지

```
❌ 매장명을 입력해주세요
❌ 업종을 선택해주세요
❌ 대표자명을 입력해주세요
❌ 전화번호 형식이 올바르지 않습니다 (예: 010-1234-5678)
```

---

### Step 4: 가입 완료

```
[✅ 체크 아이콘]

가입 완료! 🎉
```

**완료 메시지**

```
환영합니다, [매장명]님!

✅ 회원가입이 완료되었습니다

선택하신 플랜: [베이직 플랜]
무료 체험 기간: 7일 (2026-02-18까지)

💡 무료 체험 기간 동안
   선택한 플랜의 모든 기능을 사용하실 수 있습니다
   체험 종료 후 자동 과금되지 않습니다
```

**다음 단계 안내**

```
다음 단계로 빠르게 시작하세요!

1️⃣ 디자이너/직원 등록하기
2️⃣ 시술/서비스 메뉴 추가하기
3️⃣ 예약 페이지 공유하기

[대시보드로 이동 →]
[튜토리얼 보기]
```

## 🔑 비밀번호 찾기 페이지 (`/forgot-password`)

### 페이지 제목
```
비밀번호 찾기
```

### 안내 메시지
```
가입하신 이메일 주소를 입력해주세요.
비밀번호 재설정 링크를 보내드립니다.
```

### 이메일 입력 폼

```
─────────────────────────────
이메일
[                           ]
─────────────────────────────

[재설정 링크 보내기]

[로그인으로 돌아가기]
```

### 발송 완료 메시지

```
✅ 이메일이 발송되었습니다!

[이메일 주소]로
비밀번호 재설정 링크를 보냈습니다.

이메일을 확인하고 링크를 클릭해
비밀번호를 재설정해주세요.

💡 이메일이 오지 않으면 스팸함을 확인해주세요
   10분 이내에 도착하지 않으면 다시 시도해주세요
```
