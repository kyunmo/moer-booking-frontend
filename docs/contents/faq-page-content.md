## 1️⃣ 시작하기

### Q1-1. moer는 어떤 서비스인가요?
```
A. moer는 소상공인을 위한 예약 관리 시스템입니다.

주요 기능
├─ 24시간 자동 예약 접수
├─ 카카오톡 자동 알림
├─ 고객 이력 자동 관리
├─ 실시간 통계 대시보드
└─ 업종별 맞춤 기능

미용실, 필라테스, 스터디카페 등
예약이 필요한 모든 업종에서 사용 가능합니다.
```

### Q1-2. 설치가 어렵나요?
```
A. 전혀 어렵지 않습니다!

5분이면 시작 가능
1. 회원가입 (1분)
2. 매장 정보 입력 (2분)
3. 디자이너/직원 추가 (1분)
4. 시술 메뉴 추가 (1분)

복잡한 설정이나 프로그램 설치 불필요
인터넷만 연결되면 바로 사용 가능합니다.

튜토리얼 영상도 제공해드려요!
```

### Q1-3. 무료 체험은 어떻게 신청하나요?
```
A. 홈페이지에서 바로 신청 가능합니다.

신청 방법
1. "무료 체험 시작하기" 버튼 클릭
2. 이메일, 비밀번호 입력
3. 매장 정보 간단히 입력
4. 플랜 선택 (베이직 또는 프로)
5. 즉시 사용 시작!

신용카드 등록 불필요
7일 동안 선택한 플랜의 모든 기능 사용
체험 종료 후 자동 과금 없음
```

### Q1-4. 무료 체험 후 자동으로 결제되나요?
```
A. 아니요, 절대 자동 결제되지 않습니다.

안심하세요
├─ 신용카드 등록 불필요
├─ 7일 체험 종료 후 자동 결제 없음
├─ 유료 전환은 직접 선택하셔야 함
└─ 체험 종료 전 이메일 알림

체험 후 마음에 들면
→ 플랜 선택 후 결제 정보 입력
→ 유료 플랜 시작

마음에 들지 않으면
→ 아무 조치 없이 7일 후 자동 종료
```

### Q1-5. 모바일에서도 되나요?
```
A. 네, 완벽하게 지원합니다!

관리자 (사장님)
├─ 스마트폰 브라우저로 접속
├─ 모든 기능 사용 가능
├─ 예약 확인/승인/통계 확인
└─ 앱 설치 불필요

고객 (예약하는 분)
├─ 모바일 최적화된 예약 페이지
├─ 카카오톡 링크로 바로 접속
├─ 터치 최적화
└─ 빠른 로딩 (3초 이내)

PC 없어도 스마트폰만 있으면 OK!
```

### Q1-6. 어떤 업종에서 사용할 수 있나요?
```
A. 예약이 필요한 모든 업종에서 사용 가능합니다.

현재 특화 업종
├─ 미용실/네일샵
├─ 필라테스/요가
├─ 스터디카페/공방
└─ 기타 예약 필요 업종

향후 추가 예정
├─ 병원/한의원
├─ 학원
├─ 헬스장/피트니스
├─ 애견 미용
└─ 상담 서비스

업종별 맞춤 기능 제공
예: 미용실 - 포트폴리오, 재방문 알림
예: 필라테스 - 수강권, 출석 체크
```

## 2️⃣ 기능

### Q2-1. 카카오톡 알림은 어떻게 설정하나요?
```
A. 매우 간단합니다!

준비물
└─ 카카오 비즈니스 계정 (무료)

설정 방법
1. 카카오 비즈니스 가입
   → https://business.kakao.com
2. 채널 만들기 (무료)
3. moer 대시보드에서 "카카오톡 연결"
4. 채널 정보 입력
5. 완료! 즉시 사용 가능

소요 시간: 약 10분
설정 가이드 영상 제공
어려우면 1:1 지원해드려요!
```

### Q2-2. 고객 예약 페이지 주소는 어떻게 되나요?
```
A. yourshop.moer.io 형식으로 자동 생성됩니다.

예시
├─ gangnam-hair.moer.io (강남 헤어)
├─ pilates-seocho.moer.io (서초 필라테스)
└─ study-cafe-hongdae.moer.io (홍대 스터디카페)

주소 변경
├─ 매장명 기준 자동 생성
├─ 원하시면 변경 가능
└─ 영문, 숫자, 하이픈(-) 사용

링크 공유 방법
├─ 카카오톡으로 전송
├─ 인스타그램 프로필에 추가
├─ 문자 메시지 발송
└─ QR 코드 생성 (제공)
```

### Q2-3. 예약이 겹치는 것을 어떻게 방지하나요?
```
A. 시스템이 자동으로 체크합니다.

시간 충돌 방지
├─ 이미 예약된 시간은 선택 불가
├─ 시술 시간 자동 계산
├─ 다음 예약 가능 시간 자동 표시
└─ 휴무 시간 자동 제외

예시
10:00 - "컷" 예약 (60분)
→ 11:00까지 자동으로 예약 불가
→ 고객은 11:00 이후 시간만 선택 가능

사람 실수 완벽 차단!
예약 겹칠 걱정 Zero!
```

### Q2-4. 고객이 직접 예약을 취소할 수 있나요?
```
A. 설정에 따라 가능합니다.

옵션 1: 고객 직접 취소 가능
├─ 예약 페이지에서 "취소하기" 버튼
├─ 취소 시 관리자에게 알림
├─ 카카오톡 자동 발송
└─ 취소 규정 설정 가능 (예: 3시간 전까지)

옵션 2: 고객 직접 취소 불가
├─ 전화로만 취소 가능
├─ 노쇼 방지에 효과적
└─ 관리자가 수동으로 취소

추천: 옵션 1 (고객 편의)
```

### Q2-5. 포트폴리오는 몇 개까지 올릴 수 있나요?
```
A. 플랜에 따라 다릅니다.

무료 플랜
└─ 5장

베이직 플랜
└─ 무제한

프로 플랜

└─ 최대 크기: 이미지 5MB, 영상 50MB

포트폴리오 활용
├─ 해시태그로 분류
├─ 고객이 포트폴리오 보고 예약
└─ 신규 고객 유입 효과
```

```
├─ 광고 표시

베이직 플랜: 29,000원/월
├─ 월 200건 예약
├─ 디자이너 3명
├─ 카카오톡 알림
├─ 광고 제거
└─ 가장 인기 있는 플랜!

프로 플랜: 59,000원/월
├─ 예약 무제한
├─ 디자이너 무제한
├─ 재방문 알림
├─ 고급 통계
└─ 중대형 매장 추천

자세한 내용: /pricing
```

### Q3-2. VAT는 별도인가요?
```
A. 네, VAT 10%가 별도입니다.

실제 결제 금액
├─ 베이직: 29,000원 + VAT 2,900원 = 31,900원
└─ 프로: 59,000원 + VAT 5,900원 = 64,900원

사업자 고객
├─ 세금계산서 발행 가능
├─ 이메일로 자동 발송
└─ 부가세 공제 가능
```

### Q3-3. 결제 수단은 무엇이 있나요?
```
A. 다양한 결제 수단을 지원합니다.

신용카드
├─ 국내 모든 카드사
├─ 체크카드 가능
└─ 법인카드 가능

계좌이체

간편결제

편하신 방법으로 선택하세요!
```

### Q3-4. 계약 기간이 있나요?
```
A. 없습니다!

월간 결제
├─ 언제든 해지 가능
├─ 위약금 없음
└─ 클릭 한 번으로 해지

연간 결제
├─ 중도 해지 가능
├─ 남은 기간 환불
└─ 20% 할인 혜택

마음 편하게 시작하세요!
```

### Q3-5. 환불 정책은 어떻게 되나요?
```
A. 공정하게 처리합니다.

7일 무료 체험 기간 내
└─ 100% 환불 (이용 기간 상관없이)

유료 전환 후
├─ 사용일 차감 후 일할 계산
├─ 예: 29,000원 플랜, 10일 사용
│   → 19,000원 환불
└─ 신청 후 5영업일 이내 처리

연간 결제
├─ 남은 개월 수만큼 환불
├─ 예: 12개월 중 3개월 사용
│   → 9개월치 환불
└─ 동일하게 5영업일 이내 처리

환불 신청: 대시보드 > 구독 관리
```

### Q3-6. 플랜을 중간에 변경할 수 있나요?
```
A. 네, 언제든 가능합니다!

업그레이드 (무료 → 베이직 → 프로)
├─ 즉시 적용
├─ 차액만 일할 계산하여 결제
└─ 1분 안에 처리 완료

다운그레이드 (프로 → 베이직 → 무료)
├─ 다음 결제일부터 적용
├─ 현재 기간은 기존 플랜 유지
└─ 별도 환불 없음

데이터는 모두 유지
제한 사항만 변경됨
```

### Q3-7. 예약 건수를 초과하면 어떻게 되나요?
```

├─ 추가 예약 가능 (경고 표시)

프로 플랜
└─ 무제한 (걱정 없음)

갑자기 차단되는 일 없어요!
```

### Q6-3. 추천인 제도가 있나요?
```
A. 네, 있습니다!

혜택
├─ 추천인 (나): 다음 달 50% 할인
├─ 피추천인 (친구): 첫 달 50% 할인
└─ 3명 추천 시: 1개월 무료

추천 방법
1. 대시보드 > 추천 코드 확인
2. 친구에게 코드 공유
3. 친구가 가입 시 코드 입력
4. 자동으로 할인 적용

제한 없음
└─ 추천 인원수 무제한

함께 사용하면 더 저렴!
```

### Q6-4. 서비스를 그만 쓰고 싶으면 어떻게 하나요?
```
A. 언제든 간단히 해지 가능합니다.

해지 방법
1. 대시보드 로그인
2. 구독 관리 메뉴
3. "구독 해지" 버튼
4. 해지 사유 선택 (선택)
5. 최종 확인
6. 즉시 해지 완료

데이터 보관
├─ 해지 후 30일간 보관
├─ 30일 내 재가입 시 복구 가능
└─ 30일 후 완전 삭제

환불
├─ 남은 기간 일할 환불
└─ 5영업일 이내 처리

떠나시는 건 아쉽지만
언제든 환영합니다!
```
