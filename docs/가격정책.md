6. 가격 정책 — 무료 + 베이직(2만원대)
이 부분이 제일 중요하니 솔직하게 조언드립니다.
현재안: 무료(50건) / 베이직 29,000원(200건) / 프로 59,000원(무제한)
검토안: 무료 + 베이직 2만원대만
제 의견: 2티어(무료+유료) 구조에 찬성합니다. 다만 가격은 조금 더 생각해봐야 합니다.
찬성 이유:

3티어는 소상공인을 헷갈리게 함. "뭘 골라야 하지?" → 이탈
초기에 10곳 확보가 목표면 단순할수록 좋음
프로 플랜 가입자는 어차피 초기에 0명 — 관리 포인트만 늘어남

가격 관련 고려사항:
비교 서비스가격특징네이버 예약무료 (결제 수수료만)네이버 생태계 락인캐치테이블매장 무료 (예약금 수수료)수수료 모델테이블매니저월 33,000원~예약 관리 특화예모 검토안월 22,000~25,000원예약+고객+통계 올인원
제안하는 구조:
무료 플랜
├ 월 30건 예약 (50건은 너무 후함)
├ 기본 예약/고객 관리
├ 광고 표시 ❌ (삭제 권장 — 아래 설명)
└ 1명 스태프

베이직 플랜: 월 20,000원 (VAT 포함 22,0000원)
├ 월 예약 무제한
├ 전체 기능
├ 카카오 알림톡
├ 통계/리포트
├ 스태프 5명
└ 30일 무료 체험
광고 표시를 빼야 하는 이유:

무료 플랜 매장의 고객이 예약 페이지에서 광고를 보면, 매장 이미지가 떨어짐
매장 사장님이 "이걸 내 고객한테 보여줄 수 없다"고 판단 → 유료 전환 안 하고 그냥 안 씀
대신 기능 제한(예약 건수, 스태프 수)으로 유료 전환 유도하는 게 효과적

22,000원 근거:

미용실 월 매출 평균 300~500만원 → 매출의 0.5% 수준이면 저항 낮음
커피 2~3잔 값 심리 = "이 정도면 써볼 만하다"
연간으로 보면 264,000원 — 부담 없는 선

나중에 올릴 수 있다:

베타 기간: 무료 → 초기 고객 확보
런칭: 22,000원 → 가격 저항 낮춤
기능 추가 후: 프로 플랜(39,000원) 도입 → ARPU 상승

요약하면, 지금은 2티어로 단순하게 가고, 22,000원 정도로 시작한 후 서비스 가치가 증명되면 프리미엄 티어를 추가하는 게 현명합니다. 소상공인은 "일단 써보고 판단"하는 경향이 강해서, 진입 장벽을 최대한 낮추는 게 10곳 확보의 핵심입니다.
