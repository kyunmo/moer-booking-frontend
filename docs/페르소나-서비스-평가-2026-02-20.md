## 2. 페르소나 1: 김소연 (신규 고객)

### 2.1 프로필
- **28세 여성**, 직장인
- **기기:** iPhone 15 (모바일 위주)
- **기술 수준:** 보통 (카카오톡, 인스타그램 일상 사용)
- **목표:** 퇴근 후 미용실 예약하고 싶음, 카카오 검색으로 매장 발견

### 2.2 사용자 여정 (User Journey)

```
시나리오: "퇴근 후 미용실 커트 예약"

Step 1: 매장 검색 (/booking)
Step 2: 매장 상세 보기 (/booking/모어헤어)
Step 3: 예약하기 클릭
Step 4: 카카오 로그인 (/booking/login)
Step 5: 프로필 등록 (/booking/profile) - 전화번호 필수
Step 6: 4단계 예약 프로세스 (/booking/모어헤어/reserve)
Step 7: 예약 완료
```

### 2.3 단계별 경험 평가

#### Step 1: 매장 검색 — 7/10
**긍정적:**
- 검색 폼이 직관적 (키워드 + 업종 + 검색 버튼)
- 매장 카드에 평점, 영업시간, 태그 표시 → 충분한 정보
- 모바일 반응형 우수 (1열 카드)

**불편사항:**
- **위치 기반 검색 없음** → "내 주변 미용실"을 찾을 수 없음. 키워드로만 검색해야 함
- 업종 필터만 있고, **지역/거리 필터 없음**
- **인기순/추천순 정렬 없음** → 평점순과 이름순만 제공
- 검색 결과가 12개 고정 → **무한스크롤 없음**, 페이지네이션만
- 매장이 많아지면 원하는 매장 찾기 어려움

#### Step 2: 매장 상세 — 8/10
**긍정적:**
- 4개 탭 (서비스, 스태프, 리뷰, 정보) 구성 직관적
- 서비스 카테고리별 그룹화 → 커트/펌/염색 쉽게 구분
- 포트폴리오 갤러리 → 스태프 실력 확인 가능
- 모바일 FAB "예약하기" 버튼 → 항상 접근 가능

**불편사항:**
- **영업 상태 표시가 눈에 잘 띄지 않음** → 현재 예약 가능한지 불명확
- **서비스 가격 범위 표시 없음** → "커트 15,000~25,000원" 같은 개요 미제공
- 리뷰 탭에서 **서비스별/스태프별 필터 없음**
- 전화번호 복사 버튼은 좋으나, **바로 전화걸기(tel:) 링크** 미제공

#### Step 3: 카카오 로그인 — 9/10
**긍정적:**
- 1클릭 카카오 로그인 → 매우 간편
- 비회원 예약 옵션 제공 → 로그인 거부감 완화

**불편사항:**
- **카카오 로그인만 지원** → 네이버/구글 미제공 (고객용)
- OAuth 리다이렉트 후 **로딩 상태가 불안정**하게 느껴질 수 있음

#### Step 4: 프로필 등록 — 6/10
**긍정적:**
- 이름, 이메일 카카오에서 자동 채움
- 전화번호 자동 포맷팅 (하이픈 자동)

**불편사항 (핵심):**
- **신규 사용자의 프로필 등록 강제 플로우가 불명확**
  - OAuth 콜백 후 `isNewUser=true`면 `/booking/profile`로 리다이렉트되어야 하지만
  - 직접 예약 페이지로 갈 수 있고, Step 3(고객 정보)에서야 경고 표시
  - **"왜 갑자기 전화번호를 요구하지?"** → 사전 안내 부족
- **전화번호 등록 후 원래 예약 페이지로 돌아가는 플로우가 매끄럽지 않음**
- 마케팅 수신 동의 위치가 프로필 페이지 깊숙이 → 가입 시 노출 부족

#### Step 5: 4단계 예약 — 7/10
**긍정적:**
- 단계별 마법사 → 복잡한 과정을 분할해 부담 감소
- 실시간 가격/시간 합계 표시
- 날짜 선택 캘린더 직관적
- 시간 슬롯 그리드 명확
- "아무나" 스태프 옵션 → 특정 스태프 없어도 예약 가능

**불편사항:**
- **Step 1에서 서비스 선택 후 Step 2로 넘어가면 서비스 변경이 번거로움** → Step 1로 완전히 돌아가야 함
- **시간 표기가 24시간제** → "14:00" 대신 "오후 2:00" 선호하는 사용자 다수
- **예상 종료 시간 미표시** → "14:00 시작, 커트 30분이면 14:30 끝나는 건가?" 계산해야 함
- **예약 가능 시간이 왜 비활성화됐는지 설명 없음** → "이 시간이 왜 안 되지?" 혼란
- Step 3 비회원 정보 입력 시 **비밀번호 없이 전화번호만으로 예약** → 보안 우려 가능
- **Step 4(확인) 화면에서 수정이 번거로움** → 특정 항목만 수정하려면 해당 Step으로 돌아가야 함

#### Step 6: 예약 완료 — 8/10
**긍정적:**
- 예약번호 표시 + 복사 버튼
- 요약 정보 카드
- "예약 확인" / "매장으로 돌아가기" 버튼

**불편사항:**
- **예약 확정 알림 없음** → 카카오 알림톡/SMS 미연동. "예약이 정말 됐나?" 불안
- **캘린더에 추가** 기능 없음 (Google Calendar, Apple Calendar)
- 예약번호를 **어디에 저장해야 하는지** 안내 부족

### 2.4 종합 만족도: 7.2/10

**핵심 개선 포인트:**
1. 위치/지역 기반 매장 검색 추가
2. 신규 고객 프로필 등록 플로우 개선 (사전 안내 + 원활한 리다이렉트)
3. 예약 시간 오전/오후 표기 + 예상 종료 시간 표시
4. 예약 확정 알림 (최소 이메일이라도)
5. 서비스 선택 후 중간 단계에서 수정 가능하도록

## 3. 페르소나 2: 박지훈 (매장 관리자)

### 3.1 프로필
- **42세 남성**, 미용실 원장
- **기기:** PC (주로 매장 내 데스크톱), Galaxy S24 (이동 중)
- **기술 수준:** 낮음 (카카오톡, 네이버 정도 사용. 관리 소프트웨어 경험 거의 없음)
- **목표:** 전화 예약 줄이고, 고객/직원 스케줄 한눈에 관리

### 3.2 사용자 여정 (User Journey)

```
시나리오: "오전 매장 오픈 → 오늘 예약 확인 → 신규 예약 처리 → 고객 관리"

Step 1: 로그인 → 대시보드 확인
Step 2: 오늘 예약 목록/캘린더 확인
Step 3: 대기 중 예약 확정 처리
Step 4: 직접 예약 등록 (전화 예약 고객)
Step 5: 고객 정보 확인/수정
Step 6: 매출/통계 확인
```

### 3.3 단계별 경험 평가

#### Step 1: 대시보드 — 8/10
**긍정적:**
- 환영 배너에 매장명 표시 → 개인화 느낌
- 퀵 액션 4개 (예약등록, 예약관리, 고객관리, 통계) → 주요 기능 바로가기
- 오늘 예약 수, 대기 중 예약, 완료 예약 통계 → 한눈에 파악
- Trial/사용량 배너 → 구독 상태 인지
- 가이드 투어 → 신규 관리자 온보딩 우수

**불편사항:**
- **실시간 알림이 더미 데이터** → NavBarNotifications에 Materio 영문 더미 데이터 6개 표시
- **알림음/진동 없음** → 신규 예약 들어와도 모름 (30초 폴링만)
- 주간 예약 차트가 **데이터 없을 때 단순 alert** → 시각적 비주얼 개선 필요
- 모바일에서 대시보드 접근 시 **좌측 메뉴가 오버레이** → 자주 열어야 해서 번거로움

#### Step 2: 예약 캘린더/목록 — 7/10
**긍정적:**
- FullCalendar 통합 → 시각적으로 우수
- 영업시간 반영 (calendarSlotMinTime/MaxTime 동적 계산)
- 상태별 색상 구분 → 대기(주황), 확정(파랑), 완료(초록), 취소(빨강)
- 사이드바 날짜 선택 (Flatpickr) → 날짜 이동 편리
- 예약 목록: 필터 + 검색 + 데이터테이블

**불편사항:**
- **다중 직원 스케줄 동시 표시 불가** → "오늘 김스타일리스트 몇 시에 비었지?" 확인 어려움
- **예약 드래그 앤 드롭 불가** → 시간 변경 시 수정 다이얼로그 열어야 함
- 캘린더 사이드바 토글 시 **300ms setTimeout** → 리사이즈 깜빡임
- **모바일에서 VDataTable 수평 스크롤** → 예약 목록이 테이블이라 작은 화면에서 불편
- **예약 한 건씩만 상태 변경 가능** → 여러 건 한 번에 "확정" 불가
- 예약 목록 **dateFilter가 시간대 고려 안 함** → 오전/오후 경계에서 오류 가능

#### Step 3: 예약 등록/수정 — 6/10
**긍정적:**
- 기존 고객 자동완성 검색
- 서비스 다중 선택
- 직원 자동/수동 배정

**불편사항 (핵심):**
- **시간 겹침(충돌) 검증 없음** → 같은 직원에게 같은 시간에 예약 2건 등록 가능
- **서비스 선택 후 예상 종료 시간 미계산** → "커트 30분 + 펌 90분 = 2시간" 계산을 머리로 해야 함
- **최소/최대 예약 가능 일자 검증 미흡** (maxAdvanceBookingDays)
- 신규 고객 등록 시 **탭 전환(기존/신규)에 따른 유효성 규칙 전환 불안정**
- **예약 수정 시 모든 필드를 다시 입력**해야 하는 느낌 → 부분 수정 불가

#### Step 4: 고객 관리 — 7/10
**긍정적:**
- VIP/단골/신규 자동 분류
- 검색 + 필터 (이름, 전화번호)
- 고객 상세: 예약 이력 탭

**불편사항:**
- **VIP/단골 판정 기준 변경 불가** → 설정에서 임계값 조정 UI 없음
- **고객 태그 관리 불완전** → 태그 추가는 있으나 태그별 필터/그룹화 미흡
- **고객 메모/특이사항 필드가 눈에 잘 안 띔** → 알레르기, 선호 스타일 등 기록 불편
- **고객 세그멘테이션 기능 부족** → "3개월 이상 미방문 고객" 필터 없음

#### Step 5: 서비스/직원 관리 — 7/10
**긍정적:**
- 서비스 카드 그리드 → 시각적으로 파악 용이
- 카테고리 관리 다이얼로그
- 직원 프로필 + 포트폴리오

**불편사항:**
- **서비스 이름 중복 검증 없음** → "커트"를 2번 등록 가능
- **서비스 가격/시간 최대값 검증 없음** → 실수로 100만원 입력 가능
- **직원 삭제 시 해당 직원이 담당하는 서비스에서 자동 제거 안 됨**
- **직원 근무 스케줄 관리 UI 부재** → 요일별 근무시간 설정 기능 없음(매장 영업시간만 있음)

#### Step 6: 통계/리뷰 — 7/10
**긍정적:**
- 5개 탭 (매출, 예약, 고객, 직원 성과, 서비스) → 포괄적
- ApexCharts 시각화

**불편사항:**
- **필터 적용이 즉시 반영 안 됨** → "검색" 버튼 클릭 필요
- **데이터 다운로드(CSV/Excel) 불가**
- **커스텀 보고서 저장 불가**
- 리뷰 답변 기능이 **완전히 구현되지 않을 수 있음**

### 3.4 종합 만족도: 6.8/10

**핵심 개선 포인트:**
1. 예약 시간 충돌 검증 (직원별 스케줄)
2. 서비스 선택 시 자동 종료 시간 계산
3. NavBar 알림 더미 데이터 제거 + 실시간 알림
4. 다중 예약 상태 일괄 변경
5. 모바일 예약 목록을 카드 뷰로 전환
6. 통계 데이터 다운로드 기능

## 4. 페르소나 3: 이수진 (재방문 고객)

### 4.1 프로필
- **35세 여성**, 기존 고객 (이전에 예약 경험 있음)
- **기기:** Galaxy Z Flip (모바일)

Step 3: 같은 매장에서 재예약
Step 4: 프로필에서 전화번호 변경

### 4.3 단계별 경험 평가

#### Step 1: 내 예약 목록 — 7/10
**긍정적:**
- 상태별 칩 필터 (전체, 대기중, 확정, 완료, 취소)
- 각 예약 카드에 매장명, 날짜, 시간, 담당자, 가격 표시
- 취소/리뷰 버튼 조건부 표시

**불편사항:**
- **과거 예약과 미래 예약이 섞여 있음** → 시간순 정렬이지만 시각적 구분 없음
- **매장명으로 필터 불가** → 여러 매장 이용 시 특정 매장 예약만 보기 어려움
- **예약 상세 다이얼로그가 너무 간소** → 서비스 상세, 메모 등 부족
- **예약 날짜 지나도 자동 완료 안 됨** → 관리자가 수동으로 "완료" 처리해야

#### Step 2: 리뷰 작성 — 6/10
**긍정적:**
- 별점 + 내용 + 담당 스태프 선택
- 감정 텍스트 표시 (별점에 따라)

**불편사항 (핵심):**
- **리뷰 작성 경로가 직관적이지 않음**
  - 내 예약 → 완료된 예약 → "리뷰 작성" 버튼 → `/booking/{slug}/review?reservationNumber=XXX`
  - 리뷰 페이지에서 **예약번호를 다시 입력해야 할 수 있음** (쿼리 파라미터 누락 시)
- **리뷰 수정/삭제 불가** → 한번 작성하면 수정 불가
- **사진 첨부 불가** → 텍스트만 가능
- 이전에 작성한 **내 리뷰 목록 페이지 없음** → 내가 쓴 리뷰를 한눈에 볼 수 없음

#### Step 3: 재예약 — 7/10
**긍정적:**
- 매장 상세 → 예약하기로 이동 가능
- 이전 고객 정보 자동 채움 (로그인 상태)

**불편사항:**
- **"이전과 같은 서비스로 재예약" 기능 없음** → 매번 서비스 다시 선택
- **이전 담당 스태프를 기억하지 않음** → 매번 다시 선택
- **즐겨찾기/찜 기능 없음** → 자주 가는 매장을 저장할 수 없음

#### Step 4: 프로필 관리 — 7/10
**긍정적:**
- 통계 표시 (예약 건수, 리뷰 건수, 가입일)
- 읽기/수정 모드 전환

**불편사항:**
- **프로필 사진 변경 불가** (카카오 프로필 의존)
- **예약 알림 설정 없음** → 알림 시간, 방법 등 커스터마이즈 불가
- **로그인 연결 관리(카카오 연결 해제) 없음**

### 4.4 종합 만족도: 6.8/10

**핵심 개선 포인트:**
1. 재예약 기능 (이전 서비스/스태프 자동 선택)
2. 즐겨찾기 매장 기능
3. 리뷰 수정/삭제, 사진 첨부
